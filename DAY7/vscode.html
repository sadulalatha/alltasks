VS Code

VS code is an IDE(Integrated Development Environment).It is an open source code editor which is developed by Microsoft. It provides tools and resources for developers with a GUI.It is used to write the code and debug.

IDE

Its an software application that provides comprehensive features to programmer and the developers for developing application or websites.

Features

            1.Open source
            2.Platform support
            3.Development capabilities
            4.Integrated terminal
            5.Extensions
            6.Debugging

Shortcuts

Shortcut Description
Ctrl+S Save the current file
Ctrl+C Copy the selected text or item
Ctrl+V Paste the copied content
Ctrl+X Cut the selected text or item
Ctrl+Z Undo the last action
Ctrl+Y Redo the previously undone action
Ctrl+A Select all content in the current file
Ctrl+Plus(+) Zoom in the editor view
Ctrl+Minus(-) Zoom out the editor view
Ctrl+Shift+Alt+Down Add cursors to multiple lines for simultaneous editing
Alt+Up Arrow Move the selected line or block of code one line up
Alt+Down Arrow Move the selected line or block of code one line up
Ctrl+Shift+P Open command pallette
Ctrl+Shift+N New window
Ctrl+Shift+W Close window
